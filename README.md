# Simple Discord Bot

## Features

- default prefix = `!`
- prefix + help
- prefix + image `image name`
- prefix + weather `city` || !погода `город`
- prefix + gif `name`
- prefix + ping
- prefix + ball
- prefix + draw `text`
- prefix + serverinfo
- prefix + setpresence `activity type, status, activity name`
- prefix + money `currency` || курс `currency`
- prefix + convert `val1` `val2`
- prefix + short `link`
